Steps til at sætte det hele op

set hpc op med vs code:

- lav enviromet:
    - python -m venv painn_env

- aktiver enviromet
    - source painn_env/bin/activate

- installer moduler:
    - pip3 install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cu121
    - pip install pandas==2.02
    - pip install matplotlib==3.7.1
    - pip install numpy==1.24.3
    - pip install tqdm==4.66.6
    - pip install torch==2.5.1
    - pip install pytorch-lightning==2.4.0
    - pip install torch-geometric==2.6.1
    - pip install rdkit

commandoer til git:
git clone <input url for repository> - kloner jeres git ned i din mappe
    - ex: git clone https://github.com/TanneSWL/PaiNN-Project.git

Resten af disse functioner kan kun bruges i git mappen PaiNN-Project (eller andre git mapper hehe)

git status # her kan du se om du har uændrede filer og om dine filer er tilføjet

git add <fil navn> # dette tilføjer en fil (gør den klar til commit)

git commit -m "<din besked>" # næste step til at skubbe filerne - 
# dette committer filerne / gemmer dine filer så du kan komme tilbage til dette punkt

git push # skubber de commitede filer op i jeres git

git pull # henter alle filer i jeres git (du må ik ha ændringer i de filer der er blevet ændrede i i gitten)
# eller du får lidt problemer

############### hvis du ikke har nogen vigtige ændringer og gerne vil lave et git pull
git stash # (påvirker alle filer) resetter din mappe til den commit version du sidst har syncroniseret med gitten
git restore <file name> # (påvirker en fil) resetter filen til den commit version du sidst har syncroniseret med gitten

############### hvis git pull giver en fejl med fast forward bla bla
git pull origin main --no-ff # dette kan fixe det pull (gør det samme som git pull)
